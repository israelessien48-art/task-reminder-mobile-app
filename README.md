# Task Reminder Mobile App

## Description
This is a mobile application developed using Flutter and Dart. The app allows users to add and remove daily tasks.

## Features
- Add tasks
- Delete tasks
- Interactive user interface
- Multiple screens

## Technologies Used
- Flutter
- Dart

## How to Run
1. Install Flutter
2. Open the project in VS Code
3. Run:
```bash
flutter run
```
